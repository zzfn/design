## ✨ 特性

- 📦 支持深色浅色模式。
- 🛡 使用 TypeScript 开发，提供完整的类型定义文件。

## 🖥 兼容环境

- 服务端渲染。
- 目前只测试过`chrome`

## 📦 安装

```bash
npm install @zzf/design --save
```

```bash
yarn add  @zzf/design
```

## 本地调试指南

> install

1. `pnpm link --global`
2. 目标项目执行`pnpm link @zzf/design --global`

> uninstall
1. 目标项目执行`pnpm unlink @zzf/design`
2. `pnpm remove @zzf/design --global`

## 本地实时预览
`pnpm run doc:dev`
